# Skill Icons

Place SVG (or PNG) icon files here. The app expects paths defined in `src/constants/icons.js` (e.g. `javascript.svg`, `react.svg`, `html5.svg`).

If an icon file is missing, the resume and project modals will show a fallback with the skill’s initials.

Suggested sources for consistent icons:
- [Simple Icons](https://simpleicons.org/) (SVG)
- [Devicon](https://devicon.dev/) (SVG)
